# はじめに

このファイルは、[moecost] のローカル版ファイルを使用するにあたり、
使用方法の説明や注意事項について、利用者に案内するために作成しています。

最初にこのファイルを読んでから使用を始めてください。

# 利用方法について

このファイルはzip圧縮された状態で配布されているはずです。
zip圧縮を解凍し、 index.html をリッチブラウザ(google Chrome等)で開ければ
サイト上で利用していた状況と同じ画面が表示されると思います。

設定値のインポート機能などもそのまま踏襲していますので、
既にインターネットのサイトで設定済みの方は該当サイトからエクスポートし、
ローカル側でインポートすれば設定の引継ぎも可能です。

# ほかサーバとの通信について

CDNサーバーから一般公開されている汎用プログラムをダウンロードする仕組みを使用しているため、
使用開始時にプログラム類をインターネット上のサーバーからダウンロードする可能性があります。
よって、完全にインターネットから隔離された状況では、このプログラムを動かすことができないかもしれません。

通信内容はプログラムのダウンロードのみで、
録した単価や検索したアイテムなどの情報を外部サーバに送信するような仕組みは構築しておりません。
また、ネットワーク版はアクセス解析を行っておりましたが、ローカル版ではそちらも外しています。

# ローカルファイルの置き場所変更についての注意事項

このプログラムはindexedDbという仕組みを使用している関係上、
ファイルを移動した場合は「別のサイト」と見なされ、これまでに登録したレシピ情報などにアクセスできなくなる可能性があります。

配置換えを行う場合は、必ず既存の設定値をエクスポート機能を用いてエクスポートし、
配置換え完了後に当該ファイルをインポートして設定引継ぎを行ってください。

# 公開停止までの新レシピ反映について

今後、web版が更新されるたびにダウンロードファイルも定期的に更新するつもりです。
その際、readme.txt最上部の更新日／本体バージョンも更新予定です。

readme.txtの最上部が変更されていない場合は、過去のファイルをダウンロードしていると思いますので、
別途キャッシュの削除などを行ったうえでダウンロードしなおしてください。
(ダウンロードページ上の更新日も参照してください。)

web公開停止後はレシピ反映は行えませんし、このファイルのダウンロードも行えません。ご注意ください。
